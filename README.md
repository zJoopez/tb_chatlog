<h1>Chatlog</h1>

A script that logs chat to a .txt file with timestamps

<h2>How it works?</h2>

Script uses chatlog.txt that it creates to script folder.
Every time you load script, it clears the file and starts logging chat messages to it.
(if you need too store some session you can take copy of the file)

(file path is defined on top of file, if you want to change it)

<h2>Controls</h2>

/ping can be used to check if script is alive
/kys tells script to kill it self and clear hooks


<h2>Installation</h2>

download the script and toss it to the script folder of the game.
you can then select it ingame from scripts section or use /ls chatlog.lua
